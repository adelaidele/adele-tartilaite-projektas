# Adeles Tartilaites baigiamojo darbo atsiskaitymas

## Reikiamos programos
- Node.js
- Naršyklė

## Instaliavimas
1. Direktorijoje ``` book-store ```, įvykdykite komandą:
  - npm install
2. Direktorijoje ``` server```, įvykdykite komandą:
  - npm install

## Projekto paleidimas
  1. Direktorijoje ``` book-store ```, įvykdykite komandą:
      - npm start
  2. Direktorijoje ``` server ```, įvykdykite komandą:
      - npm start
